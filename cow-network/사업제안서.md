# 한우 품질평가 사업 제안서

## 스토리라인: 한우 농가의 손실을 줄이고, 우리는 수익을 창출한다

"**조기 육질 평가를 통한 한우 농가의 손실 감소와 프리미엄 축산의 고도화**"

---

## Executive Summary

### 제안의 핵심

카이바이오(한농대 교수님 소유)의 **육질 조기 평가 기술**과 씨위드의 **세포 배양 검증 기술**을 결합하여, 한우 농가에게 조기 품질평가 서비스를 제공하고, 이를 통해:

1. **농가**: 저품질 소 조기 판별 → 사료비/사육비 손실 감소
2. **우리**: 저품질 소 직매입 또는 컨설팅 수수료로 수익 창출
3. **최종 유통**: 카이에프앤비(F&B 사업체)를 통한 싼 가격 매입으로 배양육용 원육 확보

### 3가지 기대효과

1. **농가의 손실 감소**
   - 저품질 소를 조기 파악하여 사육 중단 또는 사료 최적화
   - 예상 손실액 감소: 연간 수십억원대

2. **새로운 수익 모델 창출**
   - 저품질 소 판별 용역료
   - 중+고품질 소 컨설팅 수수료
   - 직매입 소를 카이에프앤비에 재판매

3. **배양육 원육 비용 절감**
   - 시중가 10%-20% 언더로 원육 확보
   - 배양육 생산 원가 경쟁력 강화

## 1. As-is → To-be: 한우 농가의 문제점 도출

### 현재 상황 (As-is)

| 영역 | 현황 | 문제점 |
|------|------|--------|
| **육질 평가 시점** | 출하 시점(30개월 후) | 저품질 소 판별이 너무 늦음 |
| **농가 손실** | 사료비, 사육비 낭비 | 조기 판별 없이 계속 투자 |
| **사료 관리** | 일관된 고가 사료 투여 | 개별 소의 특성에 맞춰 조정 불가 |
| **원육 가격** | 시중 시세 기준 | 대량 저가 매입 기회 상실 |
| **정책 방향** | 축산 고급화 추진 중 | 농가 수익성 개선 필요 |

### 목표 상황 (To-be)

| 영역 | 목표 | 기대효과 |
|------|------|---------|
| **조기 육질 평가** | 12-18개월 시점에 평가 | 저품질 소 조기 파악 |
| **농가 손실 감소** | 개별 소 최적화 관리 | 연간 수십억원대 손실 감소 |
| **컨설팅 서비스** | 중+고품질 소 사료 처방 | 프리미엄 한우 생산 지원 |
| **직매입 시스템** | 저품질 소 직매입 → 카이에프앤비 | 원육 비용 10%-20% 절감 |
| **정책 연계** | 농가 경쟁력 강화로 정책 지원 대상** | 정부 지원금 확보 가능 |

### 시장 현황 및 기회

**한우 시장 규모**
- 국내 한우 사육농가: 약 15,000호
- 연간 출하두수: 약 300만두
- 시장규모: 약 8조원대

**문제점**
- 저품질 소 비중: 약 40-50% (연간 100-150만두)
- 농가 손실: 저품질 소당 연간 500만원-1,000만원 손실
- 정부 정책: 축산 고급화, 농가 수익성 개선 강조

**우리의 기회**
- 조기 평가를 통한 농가 손실 감소 → 정책 부합성 높음
- 저가 원육 확보 → 배양육 원가 경쟁력 강화
- 기술 독점성 → 이코노믹스 및 카이바이오 기술 결합

## 2. 솔루션: 한우 조기 품질평가 플랫폼

### 핵심 솔루션의 3가지 축

#### Pillar 1: 조기 육질 평가 (카이바이오 기술)

**카이바이오의 핵심 기술**
- **기술**: 혈액/체조직을 이용한 육질 조기 예측 기술
- **평가 시점**: 12-18개월 (조기 파악 가능)
- **정확도**: 90% 이상의 예측 정확도
- **효과**: 저품질 소 판별 시 사료비 및 사육비 즉시 최적화 가능

**서비스 제공 방식**
```
농가 신청
  ↓
혈액/체조직 샘플 채취 (12-18개월령)
  ↓
카이바이오 분석 (1-2주)
  ↓
육질 등급 예측 리포트 제공
  ↓
농가 의사결정 (계속 사육 vs 조기 출하 vs 사료 변경)
```

#### Pillar 2: 세포 배양을 통한 검증 (씨위드 기술)

**씨위드의 역할**
- **기술**: 조직 샘플의 세포 배양을 통해 성장 잠재력 검증
- **프로세스**: 조직 샘플 → 세포 배양 → 성장 지표 분석
- **결과**: 카이바이오 평가의 신뢰성 과학적 입증

**기대효과**
- 조기 평가의 신뢰도 향상
- 기술 축적으로 모델 고도화
- 향후 AI 기반 예측 모델 개발 가능

#### Pillar 3: 사료 컨설팅 및 직매입 (우리의 비즈니스)

**비즈니스 모델**
```
저품질 소 판별
  ↓
선택지 1) 농가에 컨설팅 제공 (사료 변경 권고)
선택지 2) 우리가 직매입 → 카이에프앤비에 재판매
  ↓
수익 창출 (용역료 또는 매매차익)
```

## 3. 기술: 원천기술과 플랫폼 구성

### 기술 Pillar 1: 카이바이오의 육질 조기 평가 기술

**기술 개요**
- **개발처**: 한농대 교수님 소유 (카이바이오)
- **원리**: 혈액 성분 및 생리지표를 통한 육질 예측
- **정확도**: 90% 이상
- **비용**: 검사 건당 추정 10-20만원

**기술 우위**
- 국내 유일의 상용화된 조기 육질 평가 기술
- 농가 현장에서 쉽게 샘플 채취 가능
- 빠른 분석 결과(1-2주)

### 기술 Pillar 2: 씨위드의 세포 배양 검증 기술

**기술 개요**
- **원리**: 조직 샘플에서 세포 추출 → 배양 → 성장 지표 측정
- **역할**: 카이바이오 평가의 검증 및 고도화
- **자산**: 미세조류 기반 배양액, 3차원 배양 기술

**기술 활용**
- 육질 조기 평가 모델의 신뢰성 입증
- 향후 AI 기반 예측 모델 개발 기초 데이터 축적
- 배양육용 원육 특성 분석

### 기술 Pillar 3: 통합 평가 플랫폼

**One-Stop 서비스 구조**
```
농가 신청
  ↓
카이바이오: 혈액/체조직 기반 1차 평가
  ↓
씨위드: 세포 배양 검증 (샘플 선별)
  ↓
최종 육질 등급 예측 + 컨설팅 리포트
  ↓
농가 의사결정 지원
```

**차별화 요소**
- 물리적 검사 (카이바이오) + 생물학적 검증 (씨위드)의 결합
- 단순 예측이 아닌 과학적 근거 제시
- 향후 고도화를 위한 데이터 축적

## 4. 비즈니스 모델 1: 저품질 소 판별 및 직매입

### 비즈니스 모델 1-1: 저품질 소 조기 판별 용역 (B2B)

**가치제안**
- 농가에게 조기 육질 평가 서비스 제공
- 저품질 소 판별 시 손실 최소화 전략 컨설팅
- 투명하고 과학적인 근거 제시

**서비스 구성**

| 서비스 | 가격 | 포함 내용 |
|--------|------|---------|
| **기본 평가** | 15만원/건 | 혈액/체조직 검사 + 1차 리포트 |
| **심화 평가** | 25만원/건 | 위 + 세포 배양 검증 + 상세 컨설팅 |
| **패키지** | 1,200만원/년 | 연간 100두 무제한 평가 (농가당) |

**경제학 (2025년 기준)**

| 항목 | 수치 |
|------|------|
| **평균 검사 가격** | 20만원/건 |
| **원가 (카이바이오+씨위드)** | 8만원/건 |
| **Gross Margin** | **60%** |
| **목표 검사량** (연 1,000건) | 2,000만원 |
| **목표 검사량** (연 5,000건) | 1억원 |

### 비즈니스 모델 1-2: 저품질 소 조기 직매입 (카이에프앤비 경유)

**가치제안**
- 카이바이오의 조기 평가 기술로 저품질 소 판별
- 카이에프앤비가 농가로부터 20% 할인으로 직매입
- 카이에프앤비가 10% 마진을 붙여서 씨위드(우리)에 판매
- 우리는 저가 원육을 배양육 생산에 활용

**수익 구조 (Profit Per Head 계산)**

#### 매매 메커니즘

```
농가 소유 저품질 소 (12-18개월령)
  ↓
카이바이오 평가 (조기 판별)
  ↓
카이에프앤비가 20% 할인으로 직매입
  (농가의 손실 최소화)
  ↓
카이에프앤비가 10% 마진 추가
  ↓
씨위드(우리)가 최종 구매 (저가 원육 확보)
  ↓
배양육 생산의 원가 경쟁력 강화
```

#### 세부 경제학 (Per Head 분석)

**1) 저품질 소의 특성 (12-18개월령)**

| 항목 | 수치 | 설명 |
|------|------|------|
| **예상 체중** | 500-600kg | 미성숙 단계, 아직 성장 중 |
| **예상 도체율** | 58-60% | 일반 한우 기준 58% |
| **예상 도체중** | 290-360kg | 500-600kg × 58-60% |
| **평균 도체중** | 325kg | (290+360) / 2 |

**2) 시중가 (기준)**

| 항목 | 계산식 | 수치 |
|------|--------|------|
| **시중 생우 가격** | - | 1,600원/kg (일반 저품질 기준) |
| **예상 생우 시중가** | 500kg × 1,600원 | **800만원/두** |

**3) 카이에프앤비의 구매가 (농가로부터 20% 할인)**

| 항목 | 계산식 | 수치 |
|------|--------|------|
| **시중가** | - | 800만원 |
| **할인율** | -20% | -160만원 |
| **카이에프앤비 구매가** | 800만원 - 160만원 | **640만원/두** |

**4) 카이에프앤비의 판매가 (우리에게, 10% 마진)**

| 항목 | 계산식 | 수치 |
|------|--------|------|
| **카이에프앤비 구매가** | - | 640만원 |
| **마진율** | +10% | +64만원 |
| **카이에프앤비 판매가 (우리 구매가)** | 640만원 + 64만원 | **704만원/두** |

**5) 우리의 비용 절감 효과**

| 항목 | 계산식 | 수치 |
|------|--------|------|
| **시중 원육가** | - | 800만원 |
| **우리 구매가** | - | 704만원 |
| **절감액** | 800만원 - 704만원 | **96만원/두** |
| **절감율** | 96만원 / 800만원 | **12%** |

**6) 배양육 원가 절감 효과**

| 항목 | 계산식 | 수치 |
|------|--------|------|
| **배양육 원가 중 원육 비중** | - | 30-40% |
| **원육 절감율** | - | 12% |
| **배양육 최종 원가 절감** | 12% × 35% (중간값) | **4.2%** |
| **배양육 경쟁력** | 시중 배양육 대비 저가 확보 | **우위** |

**7) 연간 목표 수익 (500두 기준)**

| 항목 | 계산식 | 수치 |
|------|--------|------|
| **원육 절감액** | 96만원/두 | 96만원 |
| **연간 두수** | 500두 | 500두 |
| **총 절감액 (원가 절감)** | 96만원 × 500 | **4.8억원** |
| **주요 가치** | - | **배양육 원가 경쟁력** |

**추가 수익 모델**

- **평가 용역료 (카이바이오 협력)**: 조기 평가 서비스 수수료
- **컨설팅 수수료**: 저품질 소 판별 후 사료 변경 권고
- **사료사 제휴**: 추천 사료 판매 수수료
- **카이에프앤비 장기 계약**: 안정적인 물량 확보로 가격 인하 협상

#### 우리(씨위드)의 역할과 가치

**역할**:
1. 카이바이오 기술로 정확한 조기 평가
2. 카이에프앤비와 파트너십을 통한 저가 원육 확보
3. 배양육 생산의 원가 경쟁력 강화

**가치**:
- 농가: 조기 판별로 손실 최소화
- 카이에프앤비: 안정적인 매입처로서 수익 창출
- 우리: 12% 저가 원육으로 배양육 경쟁력 강화

#### 주요 가정 및 리스크

**강점**
- 카이바이오 조기 평가 + 카이에프앤비 물량 확보 = Win-Win-Win
- 12% 저가 원육으로 배양육 시장 진입 시 가격 경쟁력
- 농가 손실 최소화 → 정부 정책 부합성 높음

**위험 요소**
- 도체율 변동: ±2% = ±6.4만원 영향
- 시장 가격 변동: ±100원/kg = ±32만원 영향
- 계절적 수급 변동: 여름철 가격 하락, 겨울철 가격 상승
- 카이에프앤비 물량 공급 안정성

**대응 전략**
- 카이에프앤비와 연간 물량 계약 체결 (안정성 확보)
- 월별 구매 계획으로 계절 변동성 분산
- 추가 물량처 확보 (대형 축산업체 등)
- 조기 평가 정확도 98% 이상 유지

### 비즈니스 모델 1-3: 배양육용 원육 비용 절감

**효과**
- 저가로 구매한 저품질 소 원육 → 배양육 생산에 활용
- 배양육 원가 구성에서 원육 비용: 약 30-40%
- 10%-20% 가격 인하 시 배양육 최종 원가 3%-8% 감소

**경제학**

| 항목 | 효과 |
|------|------|
| **원육 단가 절감** | 10%-20% |
| **배양육 원가 구성 중 원육 비중** | 30%-40% |
| **최종 원가 절감** | 3%-8% |
